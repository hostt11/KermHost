Soumettre un Bot

Comment ajouter votre bot ?

Étape 1 : Créer le fichier kerm.json
Créez un fichier kerm.json à la racine de votre repository GitHub avec la structure requise. Voir l'exemple
Étape 2 : Définir les variables d'environnement
Dans le fichier kerm.json, définissez toutes les variables d'environnement nécessaires à votre bot dans la section "env". Chaque variable doit avoir une description et une valeur par défaut si nécessaire.
Étape 3 : Soumettre votre bot
Remplissez le formulaire ci-dessous avec le lien GitHub de votre bot et les informations requises. Notre équipe examinera votre bot sous 24-48 heures.
Récompenses
Pour chaque déploiement de votre bot par un autre utilisateur, vous recevrez 10 coins. Plus votre bot est populaire, plus vous gagnez de coins !
